# Probes

The **Probes** section lists 0DIN's curated **probe packs** — drop-in extensions for popular open-source AI security tools (such as garak) that add 0DIN-disclosed jailbreak coverage to your existing scanning workflow.

The side nav shows one entry per supported probe product. Each entry opens a dedicated page for that product.

## Per-Product Page

Each probe product page has a header with the **Product name** and a tagline describing the pack as an extension of the upstream tool's vendor (e.g. *"Extend NVIDIA's open-source security tools with 0DIN's curated probe packs"*).

## Available Downloads

Every released version of the probe pack is listed with:

- **Version** identifier
- **Release notes** (when present)
- **Download** button that pulls the artifact directly
